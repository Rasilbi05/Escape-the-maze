# Escape-the-maze
Este proyecto fue desarrollado como parte de la asignatura Diseño Basado en Microprocesadores para la Universidad de Cádiz (UCA). El objetivo principal es aplicar el conocimiento obtenido en la asignatura sobre el desarrollo de un programa ejecutado en un microprocesador LPC-4088 y utilizar sensores externos.

## Tecnologías utilizadas 🚀
Se han usado principalmente las siguientes tecnologías:
* ***Hardware*** 🖲️
  * Microcontrolador LCP-4088
  * Módulo de foto-resistencia KY-018
  * Módulo joystick de eje XY-KY-023
  * Pantalla: **Embedded Artists LCD Expansion Board**
* ***Software*** 💻
  * IDE: **Keil uVision5**
  * Lenguaje principal: **C**
## Configuración del Entorno 🛠️
1. Instalar **Keil µVision5**.
2. Instalar el Pack de dispositivos **Keil::LPC4000_DFP**.
##  Compilación y carga 🧩
1. Abrir el proyecto 'project/proyecto.uvprojx'.
2. Pulsar 'ctrl + F7' (Translate), y luego 'F7' (Build) para compilar.
3. Conectar la placa a través de una conexión como **J-link**, y pulsar 'F8' (Download).
